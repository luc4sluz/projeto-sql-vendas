# Projeto SQL - Análise de Vendas de Medicamentos

## Objetivo
Simular análise de vendas de medicamentos em diferentes farmácias, conectando com o projeto FarmaBot.

## Estrutura do banco
- Farmácias
- Medicamentos
- Vendas

## Análises futuras
- Medicamento mais vendido
- Faturamento por farmácia
- Categoria mais vendida
- Comparação de preços

---

## Possível Aplicação Real

Este projeto simula a base de dados que pode ser utilizada em um sistema de comparação de preços de medicamentos (FarmaBot), permitindo análises como:

- Identificação de medicamentos mais vendidos
- Comparação de faturamento entre farmácias
- Monitoramento de categorias mais demandadas
